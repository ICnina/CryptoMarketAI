# Crypto Market AI Analyst

An automated Data Pipeline and AI Reporting tool that fetches live cryptocurrency data, cleans the dataset, and generates a factual, zero-hallucination "Morning Brief" using Llama 3.1.

## Project Overview
This project demonstrates a complete automated workflow—from API data ingestion to AI-generated analytics. It is designed to provide traders or financial analysts with a quick, objective overview of the crypto market's daily top movers, completely free from AI hallucinations or unverified rumors.

### Key Features & Technical Highlights
* **Automated Data Ingestion:** Fetches real-time market data for top cryptocurrencies.
* **Data Engineering & Cleaning (ETL):** Utilizes `pandas` to dynamically filter out stablecoins (e.g., Tether, USDC) and pegged assets, ensuring only true market movers are analyzed.
* **Algorithmic Sorting:** Pre-calculates the absolute top 3 winners and top 3 losers using Python, shifting the mathematical heavy-lifting away from the LLM to guarantee 100% accuracy.
* **Advanced Prompt Engineering:** Uses the Groq API (Llama 3.1) with strict system instructions and a low temperature (`0.1`) to force the AI into a strictly factual, analytical role.
* **Secure Configuration:** Implements `.env` for secure API key management.

## Tech Stack
* **Language:** Python 3.x
* **Data Processing:** Pandas
* **AI/LLM:** Groq API (Llama 3.1 8b-instant)
* **Environment Management:** `python-dotenv`

## How to Run the Project

### 1. Prerequisites
Make sure you have Python installed on your machine. You will also need a free API key from [Groq](https://console.groq.com/).

### 2. Installation
Clone this repository and install the required dependencies:

```bash
# Clone the repository
git clone https://github.com/ICnina/ebook-scraper-ai.git
cd ebook-scraper-ai 

# Install required packages
pip install requests pandas groq python-dotenv
```

### 3. Environment Setup
Create a `.env` file in the root directory of the project to securely store your API key. 

```env
# Inside your .env file
GROQ_API_KEY=your_actual_api_key_here
```
*(Important: The `.env` file is included in `.gitignore` to prevent leaking the API key).*

### 4. Running the Pipeline

**Step 1: Fetch and Clean Crypto Data**
Run the main ingestion script to fetch live market data, filter out stablecoins/pegged assets, and output Crypto_top20.csv:

```bash
python3 main.py
```

**Step 2: Generate AI Analysis**
Run the analysis script to parse the pre-calculated top movers, pass the data to Llama 3.1, and save the report:

```bash
python3 ai_analysis.py
```

## Project Structure
*   `main.py` - Fetches market data via REST API, cleans stablecoins, and saves the top assets to CSV.
*   `ai_analysis.py` - Loads CSV data, isolates top 3 winners/losers, formats the prompt, and queries Llama 3.1 via Groq.
*   `Crypto_top20.csv` - Cleaned, dynamic dataset generated by main.py.
*   `crypto_morning_report.txt` - Final AI-generated morning brief report.
*   `.env` - Environment variables storing secret API keys (untracked).
*   `.gitignore` - Prevents sensitive credentials and temporary files from pushing to GitHub.
